## Nuances

1. **ES première** : le site annonce un **futur** programme (maths intégré, BO 02/04/2026). Pour **juin 2026**, le BO **22/06/2023** reste le bon socle ES ; le candidat avec spé maths n’utilise pas le module maths intégré ES de toute façon.
2. **Noms de fichiers** : `00b_..._BO2019.pdf` contient en réalité le BO **2023** (libellé historique du projet) — le contenu PDF est correct.
3. **Contrôle académique** : l’actualité réglementaire des BO est confirmée ; le **périmètre précis des sujets** (ponctuelles fin de cycle) reste à valider au rectorat si besoin.
