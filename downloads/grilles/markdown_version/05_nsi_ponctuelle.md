# NSI — notation (évaluation ponctuelle, spé 1re)

**Source :** `../pdf_version/02_specialite_abandonnee_1re.pdf` (section NSI)

## Barème (pas de grille qualitative par question)

| Élément | Règle |
|---------|--------|
| Format | 7 parties × 6 QCM = **42 questions** |
| Bonne réponse | **+1 point** |
| Fausse / absente / multiple | **0** |
| Transformation | `note = points_obtenus × 20 / 42` |
| Durée | 2 h |
| Calculatrice | **interdite** |

## Rubriques (7 thématiques programme)

1. Représentation des données  
2. Traitement de données en tables  
3. Web  
4. Architectures matérielles et systèmes  
5. Langages et programmation  
6. Algorithmique  
7. *(Histoire transversale — intégrée aux autres)*

Voir aussi `../../syntheses/nsi_7_rubriques.md`.

## Grille détaillée

**Aucune grille d’appréciation par compétence** pour le QCM : notation **automatique** à partir des réponses. Les annexes « grille » de la NS concernent surtout les **autres spécialités** (LLCER oral, arts, etc.), pas NSI.
