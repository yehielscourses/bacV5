# Grilles d'évaluation et de notation — bac général 2026

**Important :** ce dossier distingue les **vraies grilles** (tableaux critères → niveaux ou scores CECRL → note) des **barèmes** (10+10, /20 global, points par exercice) des seules notes de service.
