# Entraînement — sujets types et annales

Dossier pour déposer localement les sujets obtenus via l’académie ou Eduscol (non inclus dans le dépôt par défaut).

## Structure suggérée

```
sujets/
├── hg/          # Ponctuelle HG fin de cycle
├── es/          # Ponctuelle ES
├── nsi/         # Banque QCM NSI
├── francais/    # Sujets zéro / types
├── pc/
├── maths/
└── lva/         # Audio CO si fourni
```

Le répertoire `sujets/` est ignoré par git (voir `.gitignore` à la racine).
